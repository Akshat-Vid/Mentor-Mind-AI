# 🎓 MentorMind AI

## 📖 Project Overview
MentorMind AI is an intelligent student performance prediction system that combines Machine Learning and Generative AI to predict student exam scores and generate personalized academic guidance.

## ✨ Features
- 📊 Student Performance Prediction
- 🤖 AI Academic Report Generation
- ⚠️ Risk Level Detection
- 📈 Machine Learning Model Comparison
- 🌐 Interactive Gradio Interface

## 🛠️ Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Google Gemini API
- Gradio

## 🤖 Machine Learning Models
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- XGBoost Regressor

## 🧠 Generative AI
Google Gemini generates:
- **Performance Summary**
- **Strengths**
- **Weaknesses**
- **Knowledge Application**
- **Personalized Improvement Suggestions**

## 📂 Dataset
StudentPerformanceFactors.csv

## ▶️ How to Run
1. Open `MentorMind_AI.ipynb` in Google Colab.
2. Add your Gemini API key using Colab Secrets.
3. Run all cells.
4. Launch the Gradio interface.
5. Enter student details and click **Predict**.

## 📸 Project Output
The application predicts the student's score, identifies the risk level, and generates a personalized AI-powered academic report.
